sass-settings
=============

Start a new project with same SASS structur and functions.

# Require install
* ruby
* sass
* compass

# To used
* create any projekt
* create folder sass `$ mkdir sass`
* `cd sass`
* add `https://github.com/CoHyper/sass-settings` as submodules insert the SASS folder
* rename `ch_config.rb` to `config.rb`
